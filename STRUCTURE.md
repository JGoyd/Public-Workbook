# Structure

This is a personal research workbook. Entries land by day.

## Layout

- `workbook/outputs/YYYY/MM/DD/<slug>/` - one folder per entry, dated.
- `workbook/topics/<slug>.md` - rollup per topic across dates.
- `workbook/indexes/` - master and per-topic indexes.
- `workbook/daily/` - short daily notes.
- `workbook/templates/` - entry skeletons.
- `workbook/_meta/` - build notes and operational files, not research.

## Entry Shape

Each entry folder has a `README.md` and any artifacts it references: reports, tables, rules, exports, raw lists, screenshots, or other files. New entries should start with frontmatter and then follow the template in `workbook/templates/entry_template.md`.

## Indexes

The Daily Log and By Topic tables in the root README are generated by `scripts/update-indexes.py`. The script also rewrites the master indexes, topic indexes, and CSV index files.

Run:

```powershell
python scripts/update-indexes.py
python scripts/update-indexes.py --check
```

Do not edit generated index tables by hand unless you are also updating the script.
