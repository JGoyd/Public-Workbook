What RemoteRAT Does

A business-invitation email passes every authentication check and lands in the victim inbox. When the victim clicks, the RATtrap JScript dropper executes via Windows Script Host. Within seconds, a Telegram bot controlled by the attacker receives the victim machine name and username.

RATtrap then installs ScreenConnect and registers the device under the attacker's remote management console. If ScreenConnect is removed, LogMeIn Resolve is already running as a second persistent channel. Both are legitimate signed applications — antivirus does not flag either one.

The attacker can see the screen, control the mouse and keyboard, transfer files, and reconnect at any time. ScreenConnect has iOS and Android clients, giving an operator a potential bridge from the Windows foothold to mobile devices on the same network. This technique is documented as T1219 (Remote Access Software) in MITRE ATT&CK.
